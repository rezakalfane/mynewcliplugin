# Usage
<!-- usage -->
```sh-session
$ npm install -g mynewcliplugin
$ oclif-example COMMAND
running command...
$ oclif-example (-v|--version|version)
mynewcliplugin/0.0.0 darwin-x64 node-v14.15.0
$ oclif-example --help [COMMAND]
USAGE
  $ oclif-example COMMAND
...
```
